**To-Do List**

Overview
Simple full-stack To-Do list where users can create, update, delete, and search tasks.
The project is built using React for the frontend and Spring Boot for the backend, with MySQL used for data storage.

**Tech Stack**

* Frontend: React.js
* Backend: Spring Boot (Java)
* Database: MySQL
* Tools: IntelliJ IDEA, VS Code, Git, GitHub

## Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Search tasks
* Persistent storage using MySQL
* REST API communication between frontend and backend
