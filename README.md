# 📝 To-Do Full Stack App

## 📌 Project Description
This is a full-stack To-Do application where users can add, edit, delete, and search tasks. It is built using React for the frontend and Spring Boot for the backend with MySQL database integration.

---

## 🚀 Tech Stack
- Frontend: React.js
- Backend: Spring Boot (Java)
- Database: MySQL
- Tools: IntelliJ IDEA, VS Code, Git, GitHub

---

## ✨ Features Implemented
- Add new tasks
- Edit existing tasks
- Delete tasks
- Search tasks
- Store tasks in MySQL database
- REST API integration between frontend and backend
